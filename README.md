Django Full-Featured Blog Application

A complete, full-featured blog application built with Django and Python. This project implements a robust user authentication system, dynamic content management, and a clean, responsive user interface. Originally inspired by Corey Schafer's renowned Django series, this version has been customized and enhanced to demonstrate core web development principles.

🚀 Technologies Used

•
Backend: Django (Python)

•
Database: SQLite (Development) / PostgreSQL ready

•
Frontend: HTML5, CSS3, Bootstrap (for responsive design)

•
Authentication: Django's built-in Auth system (Login, Logout, Password Reset)

•
Media Handling: Image upload and processing for user profiles.

✨ Key Features

•
User Authentication: Secure registration and login system.

•
User Profiles: Custom profile pages with the ability to upload profile pictures.

•
Post Management: Full CRUD (Create, Read, Update, Delete) functionality for blog posts.

•
Dynamic Routing: SEO-friendly URLs and dynamic post detail pages.

•
Pagination: Efficiently handle large numbers of posts.

•
Responsive UI: Optimized for both desktop and mobile viewing.

🛠️ Getting Started

To get a local copy up and running, follow these simple steps:

Prerequisites

•
Python 3.x

•
pip (Python package manager)

Installation

1.
Clone the repository:

Bash


git clone https://github.com/devgabrielcd/BlogProject.git





2.
Navigate to the project directory:

Bash


cd BlogProject





3.
Create and activate a virtual environment:

Bash


python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate





4.
Install dependencies:

Bash


pip install -r requirements.txt





5.
Run migrations:

Bash


python manage.py migrate





6.
Start the development server:

Bash


python manage.py runserver





7.
Access the application at http://127.0.0.1:8000.
